Dashboard de Optimización Operativa en Transportes

Este proyecto consiste en el desarrollo de un informe interactivo para una compañía de transportes, orientado a optimizar la gestión y visualización de datos operativos.

Se aplicaron técnicas avanzadas de modelado de datos, limpieza y transformación con Power Query, junto con la creación de una tabla calendario y parámetros dinámicos para facilitar el análisis temporal.

El informe incorpora marcadores para alternar entre diferentes tipos de gráficos (líneas y columnas), así como tooltips personalizados mediante páginas ocultas que enriquecen la experiencia de usuario.

Además, se diseñó un menú de filtros dinámico para permitir una exploración intuitiva y una interacción completa entre visualizaciones.
